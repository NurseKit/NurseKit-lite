> [!abstract] En 2 mots
> Excès de production des hormones thyroïdiennes (T3/T4). La cause principale est la **maladie de Basedow** (auto-immune). Tableau : tachycardie, amaigrissement, agitation, tremblement, thermophobie. Peut provoquer une **fibrillation auriculaire**. Urgence : la **thyrotoxicose aiguë** (crise thyréotoxique) est rare mais potentiellement mortelle.

## C'est quoi ?

Excès de T3/T4 → TSH effondrée (↓↓ TSH = hyperthyroïdie) → accélération de tous les métabolismes. Femme > homme (7-10:1).

**Causes principales :**
- **Maladie de Basedow** : auto-immune, anticorps anti-récepteurs TSH (TRAK), la plus fréquente chez la femme jeune
- **Adénome toxique** : nodule thyroïdien autonome hyperfonctionnel
- **Goitre multinodulaire toxique** : plusieurs nodules autonomes, sujets âgés
- **Iatrogène** : amiodarone (riche en iode), surdosage en [[Médicaments/Hormones/Levothyrox\|Levothyrox]], produits de contraste iodés

## Comment ça se présente ?

| Signe | Description |
|-------|-------------|
| **Tachycardie** | FC > 100 bpm au repos, palpitations, FA (surtout sujet âgé) |
| **Amaigrissement** | Malgré une appétit conservé ou augmenté, fonte musculaire |
| **Tremblement** | Fin, des extrémités, permanent |
| **Thermophobie** | Intolérance à la chaleur, transpiration excessive |
| **Nervosité / agitation** | Anxiété, irritabilité, insomnie, labilité émotionnelle |
| **Exophtalmie** | Signe spécifique de Basedow (yeux saillants), peut entraîner une kératite |
| **Diarrhée / accélération du transit** | Fréquente |

## Rôle IDE

- [ ] Surveillance : FC (scope si FA), TA, poids, tremblements, état psychologique, sommeil
- [ ] Traitements antithy roïdiens de synthèse (ATS) : carbimazole (Néo-Mercazole), propylthiouracile → surveillance NFS (agranulocytose grave) + bilan hépatique
- [ ] Bêta-bloquants (propranolol) pour contrôler la tachycardie et les tremblements : surveiller FC, TA
- [ ] Iode radioactif (IRA-131) ou chirurgie (thyroïdectomie) : informer le patient sur les modalités
- [ ] Exophtalmie : soins oculaires (larmes artificielles, occlusion nocturne si lagophtalmie), avis ophtalmologique
- [ ] Éviter les produits iodés (Bétadine, produits de contraste) si possible sans avis médical → peuvent aggraver l'hyperthyroïdie

## Points de vigilance

> [!danger] Agranulocytose sous ATS
> Le carbimazole et le propylthiouracile peuvent provoquer une agranulocytose (PNN < 500) → risque infectieux vital. Surveiller NFS régulièrement. Éducation patient : tout épisode de **fièvre + mal de gorge** sous ATS = arrêt immédiat du médicament + NFS en urgence → consultation aux urgences.

> [!warning] Crise thyréotoxique (thyroid storm)
> Décompensation aiguë de l'hyperthyroïdie déclenchée par une infection, une chirurgie ou un produit iodé → hyperthermie > 40°C, tachycardie extrême (> 140), troubles neuro (agitation, délire, coma), insuffisance cardiaque. Mortalité 10-20 %. Urgence réanimatoire : propranolol IV + ATS + corticoïdes + iode de Lugol.

## Les 3 choses à retenir

1. **"TSH effondrée = hyperthyroïdie"** → TSH est le premier examen, si ↓↓ → doser [[Médicaments/Normes/T4 libre\|T4 libre]]
2. **"Fièvre + mal de gorge sous ATS = urgence"** → arrêt immédiat + NFS + urgences (agranulocytose)
3. **"FA chez un sujet âgé = chercher une hyperthyroïdie"** → hyperthyroïdie masquée très fréquente chez les > 60 ans, souvent sans signes classiques
