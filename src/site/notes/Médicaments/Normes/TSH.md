# TSH : Thyréostimuline hypophysaire

| | |
|---|---|
| **Valeur normale** | **0,27-4,2 mUI/L** |
| **Unité** | mUI/L |
| ** Seuil critique** | < 0,01 = hyperthyroïdie franche | > 10 = hypothyroïdie avérée |

> [!abstract] Astuce
> 1er marqueur à doser. TSH basse + T4L haute = hyperthyroïdie. TSH haute + T4L basse = hypothyroïdie
