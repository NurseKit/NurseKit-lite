# 25-OH Vitamine D (dosage)

|                       |                                          |                                                 |
| --------------------- | ---------------------------------------- | ----------------------------------------------- |
| **Valeur normale**    | **Optimal : 30-0 ng/mL** (75-150 nmol/L) |                                                 |
| **Unité**             | ng/mL                                    |                                                 |
| **⚠️ Seuil critique** | < 10 ng/mL = carence sévère              | > 150 ng/mL = toxicité possible (hypercalcémie) |

> [!info] Astuce
> Insuffisance : 20-30 ng/mL. Carence : < 20. Complémentation si < 30 ng/mL
