# PSA total : Antigène prostatique spécifique

| | |
|---|---|
| **Valeur normale** | **< 4 ng/mL** (< 50 ans : < 2,5 ng/mL) |
| **Unité** | ng/mL |
| ** Seuil critique** | > 10 = suspicion carcinome prostatique élevée → biopsies |

> [!abstract] Astuce
> Augmenté aussi dans : HBP, prostatite, sonde urinaire, TR récent. Vélocité PSA > 0,75 ng/mL/an = alarme
