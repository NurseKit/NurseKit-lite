# TA diastolique (PAD)

| | |
|---|---|
| **Valeur normale** | **50-89 mmHg** |
| **Unité** | mmHg |
| ** Seuil critique** | < 50 mmHg ou ≥ 110 mmHg = alarme |

> [!abstract] Astuce
> PAD ≥ 90 → HTA grade 1 selon ESC 2023
