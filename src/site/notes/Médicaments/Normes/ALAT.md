# ALAT (TGP) → Alanine Aminotransférase

> [!abstract]  Norme
> **H : < 41 UI/L | F : < 33 UI/L**

---

## Ce que c'est

L'ALAT (anciennement TGP) est une enzyme **quasi exclusivement hépatique**. Quand le foie souffre (hépatite, toxicité médicamenteuse, alcool), les hépatocytes relâchent leur ALAT dans le sang → c'est le signe d'une cytolyse hépatique. C'est la transaminase la plus spécifique du foie.

---

## Pourquoi on la mesure ?

Surveillée systématiquement sous médicaments hépatotoxiques : **méthotrexate, statines, isoniazide, paracétamol en overdose, amiodarone**. En pratique, si ALAT > 3× N sous traitement, on réévalue le médicament en cause avec le médecin.

---

## Quand c'est anormal ?

**Valeur haute → cytolyse hépatique :**
> < 3× N = cytolyse minime → souvent asymptomatique
> 3-10× N = **cytolyse significative** → hépatite virale (B, C), alcool, médicaments, NASH
> > 10× N = **hépatite aiguë sévère** → bilan étiologique urgent
> ALAT > ASAT = hépatite virale ou médicamenteuse (plutôt que alcoolique)
