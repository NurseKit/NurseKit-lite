# LDL-cholestérol (« mauvais cholestérol »)

| | |
|---|---|
| **Valeur normale** | Cible variable selon le risque CV :
- Faible risque : **< 3,0 mmol/L**
- Risque modéré : **< 2,6 mmol/L**
- Risque élevé : **< 1,8 mmol/L**
- Très haut risque (ATCD CV) : **< 1,4 mmol/L** |
| **Unité** | mmol/L |
| ** Seuil critique** | > 4,0 = hypercholestérolémie sévère. Objectif primordial dans la stratification du risque CV |

> [!abstract] Astuce
> Calculé par la formule de Friedewald (sous-estime si TG élevés). Mesure directe si TG > 4,5
