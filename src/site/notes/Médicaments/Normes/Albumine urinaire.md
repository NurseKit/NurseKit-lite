# Albuminurie / Microalbuminurie

| | |
|---|---|
| **Valeur normale** | **< 30 mg/24h** (ou < 3 mg/mmol créatinine) |
| **Unité** | mg/24h |
| ** Seuil critique** | > 300 mg/24h = macroalbuminurie (syndrome néphrotique si > 3 g/24h) |

> [!abstract] Astuce
> 30-300 mg/24h = microalbuminurie → marquer précoce néphropathie diabétique. Dosage sur urines du matin préférable
