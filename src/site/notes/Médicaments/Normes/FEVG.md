# FEVG : Fraction d'éjection VG (échographie)

| | |
|---|---|
| **Valeur normale** | **≥ 55 %** |
| **Unité** | % |
| ** Seuil critique** | < 40 % = ICFEd (dysfonction systolique) → IEC + bêtabloquant + ARM | < 30 % = indication ICD/resynchronisation |

> [!abstract] Astuce
> 40-54 % = IC légèrement altérée. ≥ 50 % avec signes IC = ICFEp (diastolique)
