# PaO₂ : Pression artérielle en O₂

| | |
|---|---|
| **Valeur normale** | **80-100 mmHg** (en air ambiant) |
| **Unité** | mmHg |
| ** Seuil critique** | < 60 mmHg = insuffisance respiratoire (critères O₂ long cours si chronique) |

> [!abstract] Astuce
> Diminue avec l'âge : PaO2 min = 102 − (0,33 × âge). Sous O₂, normaliser ne garantit pas l'adéquation tissulaire
