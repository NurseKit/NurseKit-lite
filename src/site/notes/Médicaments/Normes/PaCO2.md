# PaCO₂ : Pression artérielle en CO₂

| | |
|---|---|
| **Valeur normale** | **35-45 mmHg** |
| **Unité** | mmHg |
| ** Seuil critique** | < 35 = hyperventilation/hypocapnie | > 50 = hypercapnie → insuffisance respiratoire ventilation |

> [!abstract] Astuce
> PaCO2 basse + pH haut = alcalose respiratoire. PaCO2 haute + pH bas = acidose respiratoire
